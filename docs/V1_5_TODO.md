## 验收标准

- `npm --prefix server run probe:cebpubservice -- --query=BIM --limit=5` 能返回非空样本。
- CEB 样本应至少具备：标题、官方详情链接、发布时间、地区、公告类型、开标/截止参考、服务范围、详情来源。
- 页面展示中 CEB 列表字段不再显示为完全待核验，而是“官方列表可信”；OpenClaw/Firecrawl 成功补强后显示“深抓取详情”。
- 若详情被 WAF 拦截，系统必须降级保存列表字段，并给出明确原因。
