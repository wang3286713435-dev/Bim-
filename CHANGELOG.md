# Changelog

## v1.2.0

发布时间：2026-04-23

更新内容：

- 完成输入防抖搜索、命中关键词高亮，以及“仅标题搜索 / 全文字段搜索”切换
- 新增搜索建议、最近搜索记录与常用筛选视图保存
- 完成浅色主题第一轮收口，覆盖投标机会、数据分析、详情页、监控词、临时搜索主链路
- 优化投标机会清单的密度、排序、筛选与已截止显示控制
- 新增批量操作能力，支持批量标记跟进、归档、忽略
- 重做监控日志区块，补充健康信息、抓取状态、飞书启用状态与实时刷新
- 将后台扫描/刷新进度条收口到顶部 Tab 区，并压缩为轻量状态条
- 将定时抓取周期调整为每 2 小时一次
- 修复筛选下拉被 Hero 容器裁切的问题
- 修复深圳交易中心详情增强中联系人、联系电话、预算等字段缺失与脏值覆盖问题

## v1.1.0

发布时间：2026-04-23

- 前后端统一为单应用部署模式，后端直接托管前端构建产物
- 新增投标机会清单、详情页、项目时间轴、字段完整度与建议跟进等级
- 新增数据库模糊搜索能力
- 接入 Firecrawl 详情增强与二段深度解析队列
- 接入 OpenClaw / OpenRouter AI provider 路由
- 接入飞书群推送与飞书多维表格同步
- 固化广东、广州、深圳等来源的详情页 URL 构建与坏链接处理策略
